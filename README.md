# 历史上的今天

APP是根据输入的日期返回历史上曾在今天发生的重大事件，数据接口来源于<a href="https://www.juhe.cn/docs/api/id/63">聚合数据</a>

开发环境：Eclipse ADT、 Phonegap(cordova)

程序由客户端Web app与服务端php脚本组成，自行编译只需修改listinfo.html中的url与getinfo.php中的key即可。
